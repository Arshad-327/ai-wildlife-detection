# AI-Based Wildlife Detection System 🐾

This project is an AI-powered wildlife detection system built using YOLOv8 and Arduino. It detects animals in real-time and triggers an alert system to prevent intrusion.

## 🔧 Technologies Used

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Arduino

## 🚀 Features

* Real-time animal detection
* Detection using COCO dataset (YOLOv8)
* Arduino-based alert system
* Supports multiple animal classes (dog, cow, elephant, etc.)

## 📁 Project Structure

- python/animal_detection.py → YOLOv8 detection logic  
- arduino/arduino.ino → Arduino code
